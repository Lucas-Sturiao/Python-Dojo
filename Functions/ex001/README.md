# Função Simples de Soma

## 📝 Enunciado
Desenvolva uma função em Python para realizar a soma de dois valores numéricos. O código deve seguir estas especificações:

```python
# Exemplo de chamada esperada:
resultado = somar(10, 5)
print(resultado) # Deve exibir 15
```

**Regras:**
1.  **Definição:** Utilize a palavra-chave `def` para criar a função.
2.  **Parâmetros:** A função deve receber dois números como argumento.
3.  **Retorno:** Utilize obrigatoriamente o comando `return` para enviar o resultado de volta ao chamador.
4.  **Execução:** Após definir a função, utilize-a passando dois valores de sua escolha e imprima o resultado.
5.  **Restrição:** Não utilize a função `input()`. Os valores devem ser passados diretamente como argumentos.

## 🎯 Objetivo
Praticar a sintaxe básica de definição de funções, passagem de parâmetros e o conceito de retorno de valores.
* **Categoria:** Sintaxe Python / Funções.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Definição de Funções:** Uso de `def` e nomenclatura de funções.
* **Parâmetros e Argumentos:** Passagem de dados para dentro do escopo da função.
* **Escopo e Retorno:** Uso do `return` para extrair dados processados.
* **Chamada de Função:** Invocação do bloco de código definido.
