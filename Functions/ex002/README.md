# Função com Lista (Backend Real)

## 📝 Enunciado
Simule um cenário de processamento de dados em um backend onde uma lista de números precisa ser filtrada e somada conforme regras específicas de negócio.

Dada uma lista de exemplo: `[10, 5, -3, 2, 0, 8, -1]`

**Regras:**
1.  **Definição:** Crie uma função chamada `soma_positivos(lista)`.
2.  **Iteração:** Utilize um laço `for` para percorrer os elementos da lista.
3.  **Filtro de Negativos:** Se encontrar um número negativo, utilize `continue` para ignorá-lo.
4.  **Condição de Parada:** Se encontrar o número `0`, utilize `break` para interromper o processamento imediatamente.
5.  **Retorno:** A função deve retornar o valor total da soma dos números positivos encontrados antes do critério de parada.

## 🎯 Objetivo
Aplicar conceitos de modularização (funções) integrados com controle de fluxo avançado para resolver problemas comuns de processamento de arrays/listas.
* **Categoria:** Lógica de Programação / Funções.
* **Nível:** Intermediário.

## 🛠️ Tecnologias e Conceitos
* **Definição de Funções:** Uso de `def` e passagem de coleções como argumentos.
* **Controle de Fluxo:** Uso combinado de `continue` (pular) e `break` (parar).
* **Processamento de Listas:** Iteração e filtragem de elementos.
* **Retorno de Valores:** Uso do `return` para extrair o resultado do escopo local da função.
