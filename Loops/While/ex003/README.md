# Filtro de Números com Continue e Break

## 📝 Enunciado
Desenvolva um script em Python que processe uma lista de números inteiros para realizar uma soma seletiva. O programa deve seguir estas regras de controle de fluxo:

1.  **Iteração:** Percorra uma lista de números (ex: `[10, -5, 3, 0, 8, -2]`).
2.  **Pular Negativos:** Se o número for negativo, o programa deve ignorá-lo e seguir para a próxima iteração imediatamente (utilize `continue`).
3.  **Interrupção:** Se o número for exatamente `0`, o loop deve ser encerrado imediatamente (utilize `break`).
4.  **Soma:** Apenas os números positivos encontrados antes do `0` devem ser somados.
5.  **Resultado:** Ao final, imprima o valor total acumulado.

## 🎯 Objetivo
Praticar o uso de comandos de controle de iteração (`continue` e `break`) para manipular o comportamento de loops de forma precisa.
* **Categoria:** Lógica de Programação.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Controle de Fluxo:** Laço `for`.
* **Comandos de Pulo:** `continue` para saltar iterações específicas.
* **Comandos de Parada:** `break` para encerramento prematuro.
* **Acumuladores:** Variável de soma para números positivos.
* **Lógica Condicional:** Estruturas `if` para filtrar valores.
