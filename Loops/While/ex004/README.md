# Simulação de Caixa Eletrônico (Saque Progressivo)

## 📝 Enunciado
Desenvolva um script que simule o funcionamento de um caixa eletrônico realizando saques sucessivos até o esgotamento do saldo disponível.

**Configurações e Regras:**
1.  **Saldo Inicial:** O sistema deve começar com um saldo de `1000`.
2.  **Saque Fixo:** Cada operação de saque deve subtrair exatamente `200` do saldo total.
3.  **Estrutura de Repetição:** Utilize um laço `while` que execute enquanto o saldo for positivo.
4.  **Atualização:** Após cada saque, imprima o valor do saldo restante.
5.  **Interrupção:** O programa deve parar imediatamente quando o saldo chegar a zero ou não for mais suficiente para um saque completo.

## 🎯 Objetivo
Praticar o uso de laços de repetição baseados em condições aritméticas e a atualização de variáveis acumuladoras (decremento).
* **Categoria:** Lógica de Programação.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Controle de Fluxo:** Laço `while` condicional.
* **Operadores Aritméticos:** Subtração e atribuição composta (`-=`).
* **Lógica de Comparação:** Verificação de valores maiores que zero.
* **Saída de Dados:** Monitoramento de estados variáveis com `print()`.
