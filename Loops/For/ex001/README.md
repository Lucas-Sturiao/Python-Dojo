# Soma de Pedidos Pagos

## 📝 Enunciado
Dada a seguinte lista de dicionários em Python:

```python
pedidos = [
    {"id": 1, "valor": 50,  "pago": True},
    {"id": 2, "valor": 120, "pago": False},
    {"id": 3, "valor": 80,  "pago": True},
    {"id": 4, "valor": 30,  "pago": True}
]
```

Monte o código completo que:
1. Percorra a lista `pedidos`.
2. Ignore os pedidos que não estão pagos.
3. Some os valores dos pedidos pagos em uma variável chamada `total`.

## 🎯 Objetivo
Praticar a manipulação de coleções de dados (listas e dicionários) e o uso de filtros condicionais para agregação de valores.
* **Categoria:** Lógica de Programação.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Estruturas de Dados:** Listas e Dicionários.
* **Iteração:** Laço `for` para percorrer coleções.
* **Condicionais:** Filtro de dados com `if`.
* **Operadores:** Soma acumulada.
