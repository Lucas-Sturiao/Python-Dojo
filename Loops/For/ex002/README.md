# Filtragem de Usuários Ativos e Maioridade

## 📝 Enunciado
Dada a seguinte lista de dicionários em Python:

```python
usuarios = [
    {"nome": "Lucas", "ativo": True,  "idade": 25},
    {"nome": "Ana",   "ativo": False, "idade": 30},
    {"nome": "João",  "ativo": True,  "idade": 17},
    {"nome": "Maria", "ativo": True,  "idade": 40}
]
```

Monte o código completo que siga as seguintes regras:
1. Considere somente os usuários que estão ativos.
2. Dentre os usuários ativos, conte quantos possuem idade maior ou igual a 18 anos.
3. Exiba o total encontrado.

## 🎯 Objetivo
Praticar a aplicação de múltiplas condições lógicas (filtros compostos) em estruturas de repetição.
* **Categoria:** Lógica de Programação.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Estruturas de Dados:** Listas e Dicionários.
* **Iteração:** Laço `for` para percorrer a lista de usuários.
* **Lógica Condicional:** Uso de operadores lógicos e relacionais (`if`, `and`, `>=`).
* **Contadores:** Incremento de variáveis para armazenamento de frequência.
