# Contagem de Temperaturas Elevadas

## 📝 Enunciado
Dada a seguinte lista de temperaturas registradas em graus Celsius:

```python
temperaturas = [22, 25, 19, 30, 28, 15]
```

Monte um código completo que siga as seguintes regras:
1. **Iteração:** Percorra toda a lista `temperaturas` utilizando um laço `for`.
2. **Condição:** Verifique quais temperaturas são estritamente maiores que 25°C.
3. **Contagem:** Incremente um contador para cada ocorrência que satisfaça a condição.
4. **Resultado:** Ao final do processamento, imprima o total de temperaturas encontradas.

## 🎯 Objetivo
Praticar o uso de acumuladores de contagem dentro de estruturas de repetição baseadas em listas numéricas.
* **Categoria:** Lógica de Programação.
* **Nível:** Iniciante.

## 🛠️ Tecnologias e Conceitos
* **Iteração:** Laço `for` para percorrer coleções de números.
* **Lógica Condicional:** Uso de operadores relacionais (`>`).
* **Contadores:** Técnica de incremento de variável (`contador += 1`).
* **Saída de Dados:** Exibição do resultado final com `print()`.
